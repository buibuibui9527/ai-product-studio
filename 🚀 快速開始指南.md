# 🚀 快速開始指南

這是一個 **AI 產品攝影 SaaS 平台**，可以用 AI 自動為產品照片更換背景。

## ⚡ 5 分鐘快速部署

### 1. 安裝依賴

```bash
npm install
```

### 2. 設定環境變數

複製 `.env.local.example` 為 `.env.local`：

```bash
cp .env.local.example .env.local
```

然後填入以下資訊：

```env
# Supabase (從 supabase.com 專案設定取得)
NEXT_PUBLIC_SUPABASE_URL=https://xxxxx.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=eyJhbGc...
SUPABASE_SERVICE_ROLE_KEY=eyJhbGc...

# Replicate (從 replicate.com/account/api-tokens 取得)
REPLICATE_API_TOKEN=r8_xxx...

# Lemon Squeezy (可選，用於付費功能)
LEMON_SQUEEZY_API_KEY=xxx
LEMON_SQUEEZY_WEBHOOK_SECRET=xxx
```

### 3. 設定 Supabase

1. 前往 [supabase.com](https://supabase.com) 建立專案
2. 在 **SQL Editor** 執行 `SUPABASE_MIGRATION.sql` 的內容
3. 在 **Authentication > Providers** 啟用 Google OAuth
4. 在 **Storage** 建立名為 `uploads` 的公開 bucket

### 4. 啟動開發伺服器

```bash
npm run dev
```

開啟 [http://localhost:3000](http://localhost:3000) 即可使用！

## 🆘 常見問題

### Q: 如何測試 AI 功能？

A: 需要先設定 Replicate API Token 和 Supabase。完整步驟請參考 `DEPLOYMENT_GUIDE.md`。

### Q: 可以不使用 Lemon Squeezy 嗎？

A: 可以！Lemon Squeezy 是可選的。如果不需要付費功能，可以不設定相關環境變數。

### Q: 支援哪些圖片格式？

A: 支援所有常見圖片格式（JPG, PNG, WebP 等）。
